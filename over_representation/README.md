# Over-representation Analysis of Proteomic Samples

This folder includes notebooks that find over-represented functions in a given proteomic sample. Given a biosample, we use the [NMDC-runtime API](https://api.microbiomedata.org/docs#) to access its records and data objects. More documentation on the NMDC-runtime API can be found [here](https://github.com/microbiomedata/NMDC_documentation/blob/main/docs/howto_guides/api_gui.md).
